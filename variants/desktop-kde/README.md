# Desktop KDE Variant

Feature-rich KDE Plasma desktop environment with full GUI support.

**Includes:**
- KDE Plasma desktop environment
- Firefox, Google Chrome, Chromium browsers
- Python 3.12
- VS Code desktop
- Jupyter extensions
- GTK theme support (Breeze)

**Usage:**
```bash
workspace --variant desktop-kde
```

**Access:** Connect via VNC or noVNC web interface for full desktop experience.

**Purpose:** Modern, polished desktop environment with advanced features for power users and GUI-intensive workflows.
