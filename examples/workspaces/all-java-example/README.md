# All Java Example

Comprehensive Java development environment with multiple JDK versions and build tools.

**Includes:**
- JDK versions 8, 9, 17-25 via jEnv
- Maven and Gradle build tools
- Java Jupyter notebook kernel
- Eclipse and IntelliJ IDEA (desktop variants)
- VS Code Java extensions (IDE variants)
- Lombok support

**Usage:**
```bash
cd examples/all-java-example
workspace
```

**Purpose:** Demonstrates multi-version Java development setup with all major tools and IDEs. Perfect for testing cross-version compatibility or learning different Java ecosystems.
