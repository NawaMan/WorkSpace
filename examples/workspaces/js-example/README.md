# Node.js Example

Modern Node.js/TypeScript development environment with Vite.

**Includes:**
- Node.js with npm
- TypeScript and Vite build tool
- Example web application
- Development server setup

**Usage:**
```bash
cd examples/nodejs-example
workspace
npm install          # if not already installed
npm run dev          # start development server
```

**Purpose:** Demonstrates Node.js/TypeScript development with modern tooling. Includes a Vite-based web app starter.
